# Conception pédagogique

## Principes appliqués

- **Rappel actif**: répondre avant de lire l'explication sollicite la récupération en mémoire.
- **Retour immédiat**: chaque réponse révèle correction, explication et lien facultatif.
- **Difficulté graduée**: les niveaux restent visibles sans pression temporelle.
- **Entrelacement**: les futures sessions mixtes alterneront les thèmes et représentations.
- **Espacement**: les erreurs alimentent déjà une file locale; une planification FSRS ou SM-2 viendra ensuite.
- **Sessions finies**: aucun fil infini, vie, classement ou récompense aléatoire.
- **Contrôle utilisateur**: indices, approfondissement et durée de session restent choisis.
- **Calibration**: après la correction, une confiance facultative distingue connaissance solide, bonne intuition et réponse hésitante. Elle ne change jamais le score, seulement l'espacement de la révision.
